1. Compter un caractère dans une chaîne
Programme qui affiche combien de fois un caractère apparaît dans une chaîne passée en argument.
Fonctionnalités :

Parcours de la chaîne caractère par caractère
Comparaison directe
Retourne un entier

Version étendue :

Possibilité d’ignorer la casse avec l'option -i
Utilisation de tolower()


2. Compter une valeur dans un tableau d’entiers
Programme simple qui :

Parcourt un tableau d’entiers
Compte combien de fois une valeur donnée apparaît
Affiche le résultat pour différents tableaux de test

Concepts :

Parcours d’un tableau avec un for
Passage d’un tableau à une fonction


3. Listes chaînées (struct Node)
Implémentation d’une liste simplement chaînée.
Fonctions incluses :

Création d’un nœud (malloc)
Insertion en tête de liste
Affichage de la liste
Libération complète avec free
Calcul de la somme des éléments
Calcul de la moyenne

Concepts abordés :

Pointeurs
Structures (struct)
Parcours de liste
Allocation dynamique


4. Convertisseur Celsius / Fahrenheit
Petit programme interactif proposant deux conversions :

Celsius → Fahrenheit
Fahrenheit → Celsius

Fonctionnement :

L’utilisateur choisit une option
Saisit une valeur
Le programme affiche la conversion

Formules utilisées :

C -> F : (9/5)*C + 32
F -> C : (F - 32) * 5/9

Compilation
Chaque programme peut être compilé avec :
gcc fichier.c -o programme
Puis exécuté avec :
./programme
